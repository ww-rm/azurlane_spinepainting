### 获取资源索引

请求:

`/azurlane_spinepainting/index.json`

响应:

```json
{
    "<shipName>": {
        "chName": "<舰娘中文名>",
        "hxName[可选]": "<舰娘和谐名>",
        "skins": {
            "<skinName>": {
                "chName": "<皮肤中文名>",
                "spines": [
                    {
                        "skelName": "<skel文件名>",
                        "atlasName": "<atlas文件名>",
                        "pages": ["<png1文件名>", "<png2文件名>"]
                    }
                ]
            }
        }
    }
}
```

Pydantic 解析示例:

```python
class Spine(BaseModel):
    skelName: str
    atlasName: str
    pages: List[str]

class Skin(BaseModel):
    chName: str
    spines: List[Spine]

class Ship(BaseModel):
    chName: str
    hxName: Optional[str] = None
    skins: Dict[str, Skin]

class ShipData(BaseModel):
    root: Dict[str, Ship]
```
